# Equipment Maintenance Tracker

This project is a Python Flask based web application developed as part of my SAIL VT Project.

## Project Description
The Equipment Maintenance Tracker is designed to manage equipment details and maintenance logs in an organized manner. It helps in tracking service history and maintaining records efficiently.

## Features
- Add and manage equipment details
- Maintain maintenance and service logs
- Track equipment maintenance history

## Technologies Used
- Python
- Flask
- HTML
- CSS
- SQLite

## How to Run the Project
1. Install required libraries:
   pip install -r requirements.txt
2. Run the application:
   python app.py
3. Open browser and visit:
   http://127.0.0.1:5000/
  ** Deployed in Local Host**
